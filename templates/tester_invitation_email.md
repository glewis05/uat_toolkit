## How to Test

1. **Open your tracker** using the link above
2. **Verify your name** appears in the header
3. **Read the guidance** at the top of each section
4. **Work through each test** in order
5. **Mark each test** as:
   - ✓ Pass (works as expected)
   - ✗ Fail (found an issue)
   - ! Blocked (can't test due to dependency)
   - − Skip (not applicable)
6. **Add notes** for any issues, observations, or questions
7. **Click Submit** when you're done (emails results to me)

## Important Reminders

- **Use Test Patients Only** - Make sure the "Test Patients Only" filter is ON in the application

- **Auto-Saved** - Your progress saves automatically in your browser every few seconds

- **Same Device** - Use the same browser/device each session (progress is stored locally in your browser)

- **Export Option** - You can also click "Export" to download your results as a JSON file

## Bulk Email Tips

### For Multiple Testers

1. Generate trackers first:
   ```bash
   python scripts/generate_tester_trackers.py UAT-NCCN-Q4-2025 mqeekjjz
   ```

   Note: `mqeekjjz` is the Propel Health UAT Formspree form ID

2. The script outputs all tester URLs at the end - copy those

3. Send individual emails using mail merge or manually

### Using Outlook Mail Merge

1. Create an Excel file with columns: Name, Email, TrackerURL, TestCount
2. Use Word mail merge with this template
3. Send via Outlook

### Using Gmail

1. Use a Google Sheets script or mail merge add-on
2. Or send manually - for 4 testers it's quick enough
